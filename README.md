
# Wordwall Botter

A tool used to flood wordwall leadeboards with customizable bots.


## How it works?
1. when you start the .exe file you will be asked to enter wordwall game url and some more essential info about your bots
2. the python file processes it so it can send a request to wordwall leaderboard with the bots as the payload
